Introduction

PSD files

Treble has everything ready for you.

Files can be found in special directory named design in root of bought theme. Here are the files:

  • Source.psd
  • Icon_Set_White.psd
  • Icon_Set_Dark.psd
  • Welcome_Image_Template.psd
  • Parallax01_Image_Template.psd
  • Parallax02_Image_Template.psd
  • Contact_Image_Template.psd
  • Portfolio_Thumb_Template.psd
  • Portfolio_Image_Template.psd

Main source file

We included layered psd for whole template Source.psd you can experiment with layout and colors before you start modifying css files.

Heads up! Layout and other color changes should be implemented using css files that have been included in the template files.


Welcome page

If you'd like to change the main image in the Welcome page, open Welcome_Image_Template.psd, make the necessary adjustments, and then save the file as welcome.jpg.

Heads up! Make sure that you followed safe area guidelines included in the psd file.


Parallax

Same as Welcome image apply to Parallax01_Image_Template.psd and Parallax02_Image_Template.psd, save it under appropriate names.

For example, to change the twitter feed background save the file as tw_bck.jpg. To change the features page background save the file as q_bck.jpg.


Contact page

To change the contact page background open the Contact_Image_Template.psd and save the file as map_bck.jpg.


Portfolio thumbs

To change thumbs images open Portfolio_Thumb_Template.psd. Fit your desired image into the format and save the file as 01_th.jpg.

Same for the rest of the portfolio thumbnails e.g. 02_th.jpg for second thumb, 03_th.jpg for third etc...

You can use same template to change blog thumbnails as well. Just change the name when saving images 01_th_blog.jpg, 02_th_blog.jpg.


Icon set

To change or modify the icons open Icon_Set_White.psd or Icon_Set_Dark.psd depending on the background you use. For instant change we suggest to keep the icon set image size the same as it is now.
